Thank you for using NDS-Ironmon-Tracker!

This tool REQUIRES the Bizhawk emulator v2.8 or greater. You can find the emulator at https://tasvideos.org/BizHawk/ReleaseHistory. Other versions have had issues and are not supported.

USAGE:
1. Load your ROM (Only Gen 4 NDS games(D/P/Pt/HG/SS) and Black/White are currently supported)
2. In Bizhawk, click Tools -> Lua Console
3. In the Lua Console, click the Open Script icon, and open ironmon_tracker.lua

To edit the Ironmon-Tracker configuration, open Settings.ini in any text editor.
